
# 📊 Matriz BCG (Boston) en SQL

---

## 🌟 Introducción

✨ Este proyecto muestra cómo construir una **Matriz Boston (BCG)** utilizando **SQL**, a partir de datos transaccionales.

La matriz BCG es una herramienta estratégica que permite clasificar productos o unidades de negocio en función de:

* 📈 **Crecimiento del mercado**
* 💰 **Participación relativa**

---

## 🎯 Objetivo del Proyecto

🚀 Demostrar cómo transformar datos crudos en **insights estratégicos**, utilizando únicamente consultas SQL.

Con este proyecto podrás:

* 🧠 Aplicar lógica analítica en SQL
* 📊 Clasificar productos en categorías estratégicas
* ⚙️ Preparar datos para dashboards (Power BI, Tableau, etc.)

---

## 🧩 Clasificación de la Matriz BCG

Los productos se clasifican en cuatro categorías:

* ⭐ **Estrellas (Stars)** → Alto crecimiento, alta participación
* 🐄 **Vacas Lecheras (Cash Cows)** → Bajo crecimiento, alta participación
* ❓ **Interrogantes (Question Marks)** → Alto crecimiento, baja participación
* 🐶 **Perros (Dogs)** → Bajo crecimiento, baja participación

---

## 🗄️ Estructura de los Datos

El dataset incluye las siguientes variables:

* 📅 Fecha
* 🧑‍💼 Segmento
* 🏢 Cliente
* 💳 Producto
* 🔹 Subproducto
* 👔 Director
* 🤝 Líder Comercial
* 🔢 Transacciones (tx_mes)
* 💰 TPV (tpv_mes)
* 🧾 Ticket medio
* 📊 Utilidad

---

## ⚙️ ¿Qué aprenderás?

* 📌 Cómo calcular crecimiento en SQL
* 📌 Cómo calcular participación relativa
* 📌 Cómo construir reglas de clasificación
* 📌 Cómo convertir datos en decisiones de negocio

---

## 🚀 Uso del Proyecto

1. Ejecuta el script de creación de la base de datos
2. Genera los datos simulados
3. Ejecuta las consultas de análisis
4. Clasifica los productos en la matriz BCG

---

## 📈 Resultado Esperado

Obtendrás una tabla donde cada producto estará clasificado dentro de la matriz Boston, lista para ser visualizada o analizada.
